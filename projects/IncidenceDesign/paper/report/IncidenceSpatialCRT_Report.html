Incidence-Guided Treatment Assignment Designs for Spatial Cluster Randomized Trials

Comprehensive Simulation Study Report

Author

Andrew Walther

Published

March 23, 2026

1 Introduction

This report provides a comprehensive reference for the simulation study evaluating 8 treatment assignment designs for Spatial Cluster Randomized Trials (CRTs) when outcome incidence is heterogeneous and spatially structured.

Given known historical incidence data exhibiting spatial heterogeneity, which treatment assignment strategy minimizes estimation error for the direct intervention effect \(\hat{\tau}\) across a range of spatial dependence and spillover configurations?

Application context: Sudden Unexpected Death (SUD) in North Carolina counties. The Poisson incidence mode uses an SUD base rate of 35 per 100,000, consistent with county-level analyses.

The study uses Maximum Likelihood Estimation via a spatial autoregressive model (MLE) as the primary estimator, evaluated across 2,560 parameter scenarios. Three analytical questions structure the results:

  • Q1: Which design performs best within each incidence generation mode?
  • Q2: How do design rankings change as individual parameters vary?
  • Q3: Which design wins for specific parameter combinations?

2 Study Region and Spatial Structure

The study region is a regular square lattice grid:

\[\mathcal{G} = \{s_1, s_2, \ldots, s_N\}\]

where each cluster \(s_i\) is located at integer coordinates \((x_i, y_i)\) on a \(G \times G\) grid, so \(N = G^2\). In this study, \(G = 10\) and \(N = 100\).

2.1 Spatial Contiguity

Two contiguity definitions are considered:

  • Rook contiguity: clusters sharing an edge (4-connected). \[\mathcal{N}_R(s_i) = \{s_j : |x_i - x_j| + |y_i - y_j| = 1\}\]

  • Queen contiguity: clusters sharing an edge or vertex (8-connected). \[\mathcal{N}_Q(s_i) = \{s_j : \max(|x_i - x_j|, |y_i - y_j|) = 1\}\]

2.2 Spatial Weight Matrix

Let \(\mathbf{A}\) denote the binary adjacency matrix where \(A_{ij} = \mathbf{1}[s_j \in \mathcal{N}(s_i)]\). The row-standardized spatial weight matrix is:

\[W_{ij} = \frac{A_{ij}}{\sum_{k=1}^{N} A_{ik}}, \quad i \neq j; \quad W_{ii} = 0\]

This ensures \(\sum_j W_{ij} = 1\) for each row, so spatial lags represent neighborhood averages. We construct \(\mathbf{W}_R\) (rook) and \(\mathbf{W}_Q\) (queen) weight matrices.

3 Outcome Data Generating Process

3.1 Spatial Durbin Model (SDM)

The outcome vector \(\mathbf{Y} = (Y_1, \ldots, Y_N)^\top\) is generated from the Spatial Durbin Model:

\[\mathbf{Y} = (\mathbf{I}_N - \rho\mathbf{W})^{-1}\left[\tau\mathbf{Z} + \gamma \cdot \text{Spill}(\mathbf{Z}) + \beta\mathbf{X} + \boldsymbol{\varepsilon}\right]\]

where:

Symbol Description Role
\(\mathbf{Y}\) \(N \times 1\) outcome vector Response
\(\rho\) Spatial autocorrelation parameter Controls global spatial dependence in \(\mathbf{Y}\)
\(\mathbf{W}\) \(N \times N\) row-standardized weight matrix Spatial structure (rook or queen)
\(\tau\) Direct treatment effect Estimand of interest
\(\mathbf{Z}\) \(N \times 1\) binary treatment vector \(Z_i \in \{0, 1\}\)
\(\gamma\) Spillover effect magnitude Indirect effect via treated neighbors
\(\text{Spill}(\mathbf{Z})\) \(N \times 1\) spillover term See below
\(\beta\) Effect of baseline incidence on outcome Covariate coefficient
\(\mathbf{X}\) \(N \times 1\) baseline incidence vector \(X_i \in [0, 1]\)
\(\boldsymbol{\varepsilon}\) \(N \times 1\) error vector \(\varepsilon_i \overset{\text{iid}}{\sim} N(0, \sigma^2)\)

The spatial multiplier \((\mathbf{I}_N - \rho\mathbf{W})^{-1}\) induces global spatial dependence.

Stationarity condition: \(|\rho| < 1 / \lambda_{\max}(\mathbf{W})\). For row-standardized \(\mathbf{W}\), \(\lambda_{\max} = 1\), so \(|\rho| < 1\).

3.2 Spillover Construction

\[\text{Spill}_i = \frac{\sum_{j \in \mathcal{N}(i)} Z_j}{|\mathcal{N}(i)|} = (\mathbf{W}\mathbf{Z})_i\]

Two spillover types are considered:

  1. Control-only spillover: \(\text{Spill}_i^{(\text{ctrl})} = \gamma \cdot (\mathbf{W}\mathbf{Z})_i \cdot (1 - Z_i)\)
  2. Bidirectional spillover: \(\text{Spill}_i^{(\text{both})} = \gamma \cdot (\mathbf{W}\mathbf{Z})_i\)

4 Incidence Generation Modes

The baseline incidence vector \(\mathbf{X}\) represents prior observed outcome rates. Three generation modes are considered, all producing values in \([0, 1]\).

4.1 Mode 1: Independent and Identically Distributed (iid)

\[X_i \overset{\text{iid}}{\sim} \text{Uniform}(0, 1), \quad i = 1, \ldots, N\]

No spatial structure; serves as the baseline. One config (\(\rho_X\) = N/A).

4.2 Mode 2: Spatially Correlated (SAR Filter)

\[\mathbf{X}^* = (\mathbf{I}_N - \rho_X \mathbf{W})^{-1}\mathbf{u}, \quad \mathbf{u} \sim N(\mathbf{0}, \mathbf{I}_N)\]

\[X_i = \Phi(X_i^*)\]

where \(\Phi(\cdot)\) is the standard normal CDF. Two configs: \(\rho_X \in \{0.20, 0.50\}\).

4.3 Mode 3: Poisson Count-Based

  1. Spatially correlated log-rates: \(\log \boldsymbol{\lambda}^* = \mu_0 \cdot \mathbf{1} + (\mathbf{I}_N - \rho_X \mathbf{W})^{-1}\mathbf{u}\)
  2. Poisson counts: \(C_i \sim \text{Poisson}(\lambda_i^* \cdot P_i)\) where \(P_i = 1{,}000\)
  3. Observed rates: \(R_i = C_i / P_i\)
  4. Rank-normalized: \(X_i = (\text{rank}(R_i) - 0.5) / N\)

Base rate \(\mu_0 = \log(35/100{,}000)\) for SUD. Two configs: \(\rho_X \in \{0.20, 0.50\}\).

Critical design invariant: Incidence is generated once per (mode, \(\rho_X\)) configuration and held fixed across all (\(\rho\), \(\gamma\), nb_type) scenarios.

5 Treatment Assignment Designs

Eight designs are evaluated. Each produces a binary treatment vector \(\mathbf{Z} \in \{0, 1\}^N\).

5.1 Design 1: Checkerboard (Strict Spatial Stratification)

\[Z_i = (x_i + y_i) \mod 2\]

Treatment in an alternating checkerboard pattern. Deterministic, exactly 50% treated.

5.2 Design 2: High Incidence Focus

\[Z_i = \mathbf{1}[X_i > \text{median}(\mathbf{X})]\]

Treatment assigned to above-median incidence clusters. Deterministic given \(\mathbf{X}\).

5.3 Design 3: Saturation Quadrants

Grid partitioned into four spatial quadrants. Saturation levels \(\{0.20, 0.40, 0.60, 0.80\}\) randomly assigned to quadrants. Within each quadrant, clusters randomly selected at the assigned rate.

5.4 Design 4: Isolation Buffer

Sequential treatment assignment: randomly select an available cluster, assign treatment, remove it and its neighbors from the available set. Treated clusters are never adjacent.

5.5 Design 5: 2x2 Blocking

Grid partitioned into non-overlapping \(2 \times 2\) blocks. Within each block, exactly 2 of 4 clusters randomly assigned to treatment.

5.6 Design 6: Balanced Quartiles

Clusters stratified by incidence quartile. Within each stratum, approximately half randomly assigned to treatment.

5.7 Design 7: Balanced Halves

Clusters stratified into two groups by median baseline incidence. Within each half, approximately half randomly assigned to treatment. A 2-stratum analog of Design 6.

5.8 Design 8: Incidence-Guided Saturation Quadrants

Grid partitioned into four spatial quadrants (same as Design 3). Quadrants ranked by average baseline incidence and deterministically assigned saturation levels:

Rank (by \(\bar{X}_{Q_k}\)) Saturation \(p_k\)
1 (highest) 0.80
2 0.60
3 0.40
4 (lowest) 0.20

Within each quadrant, clusters randomly selected at the assigned rate. Unlike Design 3, the saturation-to-quadrant mapping is guided by incidence data rather than random.

6 Estimation Methods

6.1 Difference in Means (DIM)

The naive estimator: \[\hat{\tau}_{\text{DIM}} = \bar{Y}_1 - \bar{Y}_0\]

DIM is biased for \(\tau\) when spillover (\(\gamma \neq 0\)) or spatial dependence (\(\rho \neq 0\)) is present.

6.2 Maximum Likelihood Estimation (MLE)

The spatial autoregressive model is estimated via maximum likelihood using spatialreg::lagsarlm():

\[Y_i = \rho \sum_j W_{ij} Y_j + \tau Z_i + \gamma \cdot \text{Spill}_i + \beta X_i + \varepsilon_i\]

MLE simultaneously estimates \(\rho\) and regression coefficients. The true spillover covariate \(\text{Spill}(\mathbf{Z})\) is included as an oracle regressor. Confidence intervals for \(\tau\) use the asymptotic covariance matrix.

MLE is the primary estimator throughout this report. DIM serves only as a naive baseline.

7.3 Per-Scenario Reproducibility

Each scenario uses a deterministic seed: \[\text{seed}_s = \texttt{digest::digest2int}(\texttt{paste}(\text{inc\_mode}, \rho_X, \text{nb\_type}, \rho, \gamma, \text{spill\_type}, d_{\text{id}}, \text{sep}=\text{"|}"))\]

This ensures results are reproducible and order-independent.

8 Evaluation Metrics

For each scenario, let \(\hat{\tau}_1, \ldots, \hat{\tau}_K\) denote the \(K\) estimates of \(\tau\) (excluding convergence failures).

  • Bias: \(\text{Bias} = \bar{\hat{\tau}} - \tau\)
  • Standard Deviation: \(\text{SD} = \sqrt{\frac{1}{K-1}\sum_{k=1}^{K} \left(\hat{\tau}_k - \bar{\hat{\tau}}\right)^2}\)
  • Mean Squared Error: \(\text{MSE} = \text{Bias}^2 + \text{SD}^2\) — primary ranking criterion
  • Coverage Probability: \(\text{Coverage} = \frac{1}{K}\sum_{k=1}^{K} \mathbf{1}\left[\tau \in \left(\hat{\tau}_{k,L},\ \hat{\tau}_{k,U}\right)\right]\)
  • Convergence Failure Rate: Proportion of iterations where MLE failed to converge

16 Conclusions and Future Work

The simulation study evaluated 8 treatment assignment designs across 2,560 parameter scenarios.

Key findings:

  1. MLE achieves near-nominal coverage. Average coverage across all scenarios is 0.89, close to the nominal 0.95 target.

  2. Design 8 is the top performer. It achieves the lowest average MSE (0.079) and coverage of 0.939.

  3. Design 1 is consistently worst. Highest MSE (0.744) and lowest coverage (0.553). Its perfect spatial alternation creates systematic confounding.

  4. Spillover type is the key moderator. Under control-only spillover, D3: Sat. Quadrants dominates. Under bidirectional spillover, D6: Balanced Quartiles becomes competitive.

  5. Design rankings are stable across incidence modes and Gamma levels. The ordering is largely consistent, and spillover magnitude does not affect design choice.

16.1 Limitations and Planned Extensions

  • Oracle MLE: The MLE model uses the true spillover covariate.
  • Fixed \(\tau\): All results use \(\tau = 1.0\). Sensitivity to varying \(\tau\) is planned.
  • Equal populations: Poisson mode uses \(P_i = 1{,}000\) for all clusters.
  • Grid sensitivity: Results use \(G = 10\) (\(N = 100\)) only.

17 Appendix A: Simulation Pseudocode

BUILD spatial grid -> coords, W_rook, W_queen, listw_rook, listw_queen
SET I_mat = diag(N)

FOR EACH incidence_mode IN {iid, spatial, poisson}:
  SET rho_X_values = {0} if iid, else {0.20, 0.50}

  FOR EACH rho_X IN rho_X_values:
    GENERATE X_matrix (N x R_o) using incidence_mode, rho_X, W_queen
    SET base_incidence = X_matrix[, 1]

    FOR EACH nb_type IN {rook, queen}:
      SELECT active W, listw for nb_type

      FOR EACH rho IN {0.00, 0.01, 0.20, 0.50}:
        COMPUTE inv_mat = solve(I_mat - rho * W)
        GENERATE Epsilon_matrix (N x R_o) ~ N(0, sigma^2)
        COMPUTE baseline_part = inv_mat %*% (X_matrix * beta + Epsilon_matrix)

        FOR EACH gamma IN {0.50, 0.60, 0.70, 0.80}:
          FOR EACH spill_type IN {control_only, both}:
            FOR EACH design_id IN {1, 2, 3, 4, 5, 6, 7, 8}:

              SET scenario_seed = hash(incidence_mode, rho_X, nb_type,
                                       rho, gamma, spill_type, design_id)
              set.seed(scenario_seed)

              GENERATE Z_matrix (N x R_d)
              FOR d IN 1:R_d:
                Z = Z_matrix[, d]
                COMPUTE spillover = W %*% Z (apply spill_type mask)
                COMPUTE trt_effect = inv_mat %*% (tau * Z + spillover)
                Y_sim = trt_effect + baseline_part
                ESTIMATE tau_hat, CI for each of R_o outcome columns

              COMPUTE Bias, SD, MSE, Coverage, Fail_Rate
              STORE results row
