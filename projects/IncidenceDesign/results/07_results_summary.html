Treatment Assignment Design Evaluation for Spatial CRTs

Simulation Study with Heterogeneous Incidence — Results Summary

Author

Andrew Walther

Published

March 22, 2026

1 Introduction

This report summarizes a simulation study evaluating 8 treatment assignment designs for Spatial Cluster Randomized Trials (CRTs) when outcome incidence is heterogeneous and spatially structured. The central research question is:

Given known historical incidence data exhibiting spatial heterogeneity, which treatment assignment strategy minimizes estimation error for the direct intervention effect \(\hat{\tau}\) across a range of spatial dependence and spillover configurations?

Application context: Sudden Unexpected Death (SUD) in North Carolina counties. The Poisson incidence mode uses an SUD base rate of 35 per 100,000 (Mirzaei et al.), consistent with county-level analyses by Gan et al. and Watson et al.

The study uses Maximum Likelihood Estimation via a spatial autoregressive model (MLE) as the primary estimator, evaluated across 2,560 parameter scenarios.

2 Mathematical Framework

2.1 Study Region and Spatial Structure

The simulation operates on a regular lattice grid \(\mathcal{G} = \{s_1, \ldots, s_N\}\) on \([1, G] \times [1, G]\), with \(N = G^2\) clusters (\(G = 10\), \(N = 100\)).

Two contiguity structures define neighbor relationships:

  • Rook contiguity: \(s_j \in \mathcal{N}(s_i)\) iff \(s_i\) and \(s_j\) share an edge (up to 4 neighbors)
  • Queen contiguity: \(s_j \in \mathcal{N}(s_i)\) iff \(s_i\) and \(s_j\) share an edge or corner (up to 8 neighbors)

The spatial weight matrix \(\mathbf{W}\) has entries \(w_{ij} = |\mathcal{N}(s_i)|^{-1}\) if \(s_j \in \mathcal{N}(s_i)\) and \(0\) otherwise (row-standardized).

2.2 Outcome Data Generating Process

Outcomes follow a Spatial Durbin Model (SDM):

\[\mathbf{Y} = (\mathbf{I} - \rho \mathbf{W})^{-1} \left[\tau \mathbf{Z} + \gamma \cdot \text{Spill}(\mathbf{Z}) + \beta \mathbf{X} + \boldsymbol{\varepsilon}\right]\]

Parameter definitions:

Symbol Role Value / Range
\(\tau\) Direct treatment effect (estimand) \(1.0\) (fixed)
\(\rho\) Outcome spatial autocorrelation {0, 0.01, 0.2, 0.5}
\(\gamma\) Spillover magnitude {0.5, 0.6, 0.7, 0.8}
\(\beta\) Incidence effect on outcome \(1.0\) (fixed)
\(\sigma\) Residual standard deviation \(1.0\) (fixed)

Spillover construction:

\[\text{Spill}_i(\mathbf{Z}) = \frac{\sum_{j \in \mathcal{N}(i)} Z_j}{|\mathcal{N}(i)|}\]

Two spillover modes are evaluated:

  • control_only: \(\gamma\) applied only to units with \(Z_i = 0\)
  • both: \(\gamma\) applied to all units regardless of treatment status

Stationarity requires \(|\rho| < 1/\lambda_{\max}(\mathbf{W})\); all swept \(\rho\) values satisfy this condition.

2.3 Incidence Generation

Three modes generate the cluster-level covariate \(X_i \in [0,1]\), each representing a different assumption about how historical incidence arises:

Mode 1 — iid Uniform: \[X_i \overset{\text{iid}}{\sim} \text{Uniform}(0, 1)\]

No spatial structure; serves as the baseline. One config (rho_X = N/A).

Mode 2 — Spatial (SAR filter): \[\mathbf{X}^* = (\mathbf{I} - \rho_X \mathbf{W})^{-1} \mathbf{u}, \quad \mathbf{u} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})\] \[X_i = \Phi(X_i^*) \in (0, 1)\]

Spatially correlated via SAR filter; probability integral transform (\(\Phi\)) ensures \(X_i\) is marginally \(\text{Uniform}(0,1)\) but spatially dependent. Two configs: \(\rho_X \in \{0.20, 0.50\}\).

Mode 3 — Poisson (count rates):

  1. Spatially correlated log-rates: \(\log \lambda_i^* = \mu_0 + [(\mathbf{I} - \rho_X \mathbf{W})^{-1} \mathbf{u}]_i\)
  2. Poisson counts: \(C_i \sim \text{Poisson}(\lambda_i^* \cdot P_i)\) where \(P_i = 1{,}000\) (equal population per cluster)
  3. Observed rates: \(R_i = C_i / P_i\)
  4. Rank-normalized: \(X_i = (\text{rank}(R_i) - 0.5) / N\)

Base rate \(\mu_0 = \log(35/100{,}000)\) mimics SUD incidence. Two configs: \(\rho_X \in \{0.20, 0.50\}\).

Critical design invariant: Incidence is generated once per (mode, \(\rho_X\)) configuration and held fixed across all (\(\rho\), \(\gamma\), nb_type) scenarios. This reflects the realistic constraint that a researcher observes historical incidence once before designing the trial; spatial model parameters affect the outcome DGP, not the incidence itself.

Designs 1 and 2 are deterministic given the incidence vector: they produce the same assignment on every resample. The simulation detects this and runs each once, replicating the result across all design resamples (saving ~25x compute).

2.5 Estimation Method

Maximum Likelihood (MLE via SAR):

Fits lagsarlm(Y ~ Z + Spill + X, listw = W) from the spatialreg package, simultaneously estimating \(\rho\), \(\tau\), and regression coefficients. The true spillover covariate \(\text{Spill}(\mathbf{Z})\) is included as an oracle regressor (i.e., the researcher is assumed to know the true spillover structure). CIs are derived from the asymptotic covariance matrix.

3.3 Evaluation Metrics

For each scenario, across \(K\) total iterations (design × outcome resamples):

\[\text{Bias} = \bar{\hat{\tau}} - \tau, \quad \text{SD} = \sqrt{\frac{1}{K-1}\sum_{k=1}^K (\hat{\tau}_k - \bar{\hat{\tau}})^2}\]

\[\text{MSE} = \text{Bias}^2 + \text{SD}^2\]

\[\text{Coverage} = \frac{1}{K}\sum_{k=1}^K \mathbf{1}\left[\tau \in (\hat{\tau}_{k,L}, \hat{\tau}_{k,U})\right]\]

3.4 Per-Scenario Reproducibility

Each scenario uses a deterministic seed: \[\text{seed}_s = \texttt{digest::digest2int}(\texttt{paste}(\text{inc\_mode}, \rho_X, \text{nb\_type}, \rho, \gamma, \text{spill\_type}, d_{\text{id}}, \text{sep}=\text{"|}"))\]

This ensures results are reproducible and order-independent: adding or removing scenarios from the parameter sweep does not affect other scenarios’ seeds.

7 Conclusions

7.1 Summary of Findings

The simulation study evaluated 8 treatment assignment designs across 2,560 parameter scenarios, spanning three incidence generation modes, two contiguity structures, 4 spatial autocorrelation levels, 4 spillover magnitudes, and two spillover types.

Key findings:

  1. MLE achieves near-nominal coverage. The average coverage across all scenarios is 0.89, close to the nominal 0.95 target. This reflects MLE’s explicit modeling of spatial autocorrelation \(\rho\), which ensures valid inference when spatial dependence is present.

  2. Design 8 is the top performer. It achieves the lowest average MSE (0.079) and a mean coverage of 0.94. Random saturation per quadrant introduces sufficient variation in spatial treatment patterns, allowing the spatial autoregressive model to cleanly separate \(\tau\) from \(\rho\).

  3. Design 1 is the consistently worst design. It achieves the highest average MSE (0.744) and the lowest mean coverage (0.55). Its perfect spatial alternation creates systematic confounding when incidence is spatially structured, inflating bias and degrading coverage.

  4. High Incidence Focus and Balanced Quartile/Strata designs are robustly competitive. These designs rank consistently well across incidence mode and parameter regimes. Incidence-stratified assignment directly addresses confounding from \(X\).

  5. Design rankings are stable across incidence modes. The ordering of designs by MSE is largely consistent across iid, spatial, and Poisson incidence configurations, suggesting the findings generalize across different assumptions about prior data structure.

7.2 Limitations and Planned Extensions

  • Oracle MLE: The MLE model uses the true spillover covariate. Non-oracle comparisons (include_spill_covariate = FALSE) remain as a planned extension.
  • Iteration count: MLE uses 250 iterations per scenario. Apparent variance inflation at high-\(\rho\) may partly reflect this count; a larger sweep would reduce Monte Carlo error.
  • Equal populations: Poisson mode uses \(P_i = 1{,}000\) for all clusters. Heterogeneous population sizes (log-normal) would more closely match NC county data.
  • Grid sensitivity: All results use \(G = 10\) (\(N = 100\)). Sensitivity to \(G = 8\) or \(G = 15\) has not been evaluated.
